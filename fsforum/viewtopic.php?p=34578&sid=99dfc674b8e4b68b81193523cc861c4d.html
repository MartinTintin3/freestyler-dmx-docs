Hey All Forum Users.

DJSupport and me have been working on a bug report system so the developer Raph and the other developers who makes some fancy plugins (like Sound 2 light by Remco, Go Monky by mattotone etc) better can keep track of the bugs.

Also. as an user you can see which issues have been found/reported (so you dont report a bug which allready is known), when they will be fixed and released in a new version, and so on.. it really is a win win win situation for us all

Now FreeStyler Bug Tracker (FsBT) is up running and people should post all the bugs there. It is easier for you and much easier for Raph. However Feel free to still ask on this forum if it really is a bug you have found.

To file a bug report you should do the following.

1) try to reproduce the issue yourself a couple of times before reporting it, so you can be specific about this in the bug report.

2) Enter The Freestyler Bug tracker by clicking on the Submit a Bug Report link placed in the "Where would you like to go?" area positioned on top on this forum.

2) Sign In
If you don't have a user, sign up (unfortunately FsBT can't retrieve the user from the FS forum)
2a) Please use the same user name as you use on the forum.
2b) Please fill out as many details in the "my Account" menu, specially the submenu "Profiles". It will make it easier and quicker to make a report and you give the developers more details about your setup

3) make sure your issue is not already reported by checking the "view issue" page. Remember to select project from project bar or the drop down menues

4) Choose the "Project" you want to make a bug report to from the Project Bar or the drop down menues

5) Choose "Report Issues"

6) Fill out the Bug report with at least the required fields
6a)Write so many details as possible as this makes the developers job easier when they shall reproduce and fix the issue.

If the report is not detailed enough and/or Developers can't reproduce the issue we will ask for feedback and more details via the bug tracker system.

If you have any question feel free to pm me.


/LJ_krede.dk
