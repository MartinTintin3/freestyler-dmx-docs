Auto Strobe Timer

  • Quote

Post by Mattotone » October 17th, 2009, 5:35 pm

auto-strobe.png
Simple timer for activating a strobe with set intensity and speed.
Interval insteps of 30seconds from 0 - 15minutes.
duration insteps of 1 second from 0 - 60 seconds.
chanels can be set by selecting the channel number and entering the new value.
V1.2
Channel names can be set be selecting and entering new value.
I.e Fog, Fan, speed, Intesity etc
v1.3
No transparent text boxes in hope to eliminate fm20.dll error, removed traces of Microsoft Forms 2.0

can be used on any two channels so not limited to just strobes, 2nd fogger etc.
if you require longer intervals, durations or channels just pm me and il send you an update.

Post by Mattotone » October 19th, 2009, 8:26 pm

start FS -> start the timer app.
then in the timmer app set the channels to the channel you want to control i.e the channel with the fogger or strobe.
set the interval/wait time and then set the duration/on time.

press start and away you go, it is set as an overide so if you have a sequence that controles the channels the app will overide them.

in vista you need to run as administrator so that windows will alow the send message commands.
