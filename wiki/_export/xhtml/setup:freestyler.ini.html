Editing the freestyler.ini file

FreeStyler uses the text file freestyler.ini to control some setup values. This can be edited in any text editor like Windows Notepad, or using FreeStyler configuration tool, a simple tool that allows editing the freestyler.ini using a graphical user interface.

Click here to Download the FS configuration tool


Set [ResetAll] to True to make FreeStyler reset itself to all its original installation values. 

[ResetAll]
reset=False
